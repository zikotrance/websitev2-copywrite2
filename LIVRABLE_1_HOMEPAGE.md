## 1. ANALYSE DE L'EXISTANT

### 1.1 Structure Actuelle (PROD-003)

| Bloc | Élément | Problème Identifié |
|------|---------|-------------------|
| 1 | Hero | H1 générique "Zero-Risk Custom Mirrors" - ne capte pas l'attention |
| 2 | Social Proof Bar | Positionné APRÈS le hero (below fold sur mobile) |
| 3 | Problem Cards | 3 cartes = 90+ mots, charge cognitive excessive |
| 4 | Solution Section | CTA secondaire "See How It Works" en compétition avec le CTA principal |
| 5 | Industry Hub | Redondant avec la navigation principale |
| 6 | Case Study Teaser | Positionné trop tôt dans le flow |
| 7 | FAQ | 5 questions = 300+ mots, trop long |
| 8 | Final CTA | Trop éloigné du haut de page |

### 1.2 Problèmes Critiques (Audit)

| Problème | Impact | Priorité |
|----------|--------|----------|
| Trust Bar below fold (mobile) | 65% du trafic mobile ne voit jamais la preuve | **P0** |
| 3 problem cards diluent le message | Charge cognitive, perte d'attention | **P0** |
| Multiple CTAs en compétition | Paralysie de décision | **P0** |
| 800+ mots par page | Dépasse la tolérance B2B (8-15 sec) | **P1** |
| H1 sans quantification | Valeur perçue faible | **P1** |

---

## 2. STRUCTURE OPTIMISÉE

### 2.1 Nouvelle Architecture (7 Blocs)

| Ordre | Bloc | Purpose | Framework |
|-------|------|---------|-----------|
| **1** | Hero + Trust Intégré | Attention + Preuve immédiate | AIDA (Attention) + MECLABS (Anxiety first) |
| **2** | CTA Inline (Desktop) | Capture high-intent | Hormozi (Effort↓) |
| **3** | Problem Block (PAS unique) | Problem + Agitate | PAS + StoryBrand (Problem) |
| **4** | Solution Overview | Plan 4 étapes | StoryBrand (Plan) + PAS (Solution) |
| **5** | Case Study Teaser | Proof | StoryBrand (Guide authority) |
| **6** | Comparison Table | Objection handling | MECLABS (Anxiety↓) + AIDA (Desire) |
| **7** | Final CTA + Micro-FAQ | Action | AIDA (Action) |

### 2.2 Réduction de Contenu

| Métrique | Avant | Après | Réduction |
|----------|-------|-------|-----------|
| Blocs | 8 | 7 | -12.5% |
| Mots | ~800 | ~500 | -37.5% |
| Questions FAQ | 5 | 3 | -40% |
| CTAs par page | 3+ | 2 (Hero + Final) | -33% |
